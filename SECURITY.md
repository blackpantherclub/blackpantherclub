Security Policy — Black Panther Club
🛡️ Versioni supportate
Versione	Supportata
main (latest)	✅ Sì
versioni precedenti	❌ No
---
📢 Segnalare una vulnerabilità
Se hai scoperto una vulnerabilità di sicurezza in questo progetto, ti chiediamo di seguire le istruzioni qui sotto per segnalarla in modo responsabile.
Come segnalare
NON aprire una Issue pubblica per vulnerabilità di sicurezza.
Invia una segnalazione privata tramite uno dei seguenti canali:
Email: [inserisci qui la tua email]
GitHub Private Security Advisory: usa la funzione Report a vulnerability di questo repository
Cosa includere nella segnalazione
Per permetterci di valutare e risolvere il problema nel più breve tempo possibile, includi:
Descrizione della vulnerabilità
Passi per riprodurla (proof of concept)
Impatto potenziale (cosa potrebbe fare un attaccante)
Eventuale suggerimento per la correzione
---
⏱️ Tempi di risposta
Fase	Tempo stimato
Conferma ricezione segnalazione	Entro 48 ore
Valutazione iniziale	Entro 5 giorni lavorativi
Rilascio della correzione	Entro 30 giorni (dipende dalla gravità)
---
🔐 Scope
Questa policy copre:
Il sito web Black Panther Club (blackpantherclub.github.io)
Il Playlist Generator e le sue API
Qualsiasi componente presente in questo repository
Non è coperto:
Servizi di terze parti (YouTube API, Google OAuth, ecc.)
Vulnerabilità nei browser o sistemi operativi degli utenti
---
✅ Divulgazione responsabile
Ci impegniamo a:
Rispondere a ogni segnalazione in modo tempestivo
Tenerti aggiornato sullo stato della correzione
Darti credito pubblico (se lo desideri) una volta risolto il problema
Non intraprendere azioni legali contro chi segnala in buona fede
Grazie per contribuire alla sicurezza del progetto. 🙏
